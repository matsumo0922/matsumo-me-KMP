# 開発用テスト

ポートフォリオ作成中に必要となるブログのサンプルです。

## 概要

このような感じで、マークダウン方式のブログや、Zen に投降したもの、Qiita に投稿したものを拾い上げてポートフォリオを作っていく予定です。基本的にはマークダウン方式が多くなると思います。

Zen や Qiita から記事を取得するために Rails バッチを動かしています。記事の順番は Rails が記事を取得して DB
に登録した時間で、必ずしも記事を執筆した時間と同じであることはないのでご注意を。

## 以下テスト

### Android

```kotlin
fun main() {
    val numbers = arrayOf(1, 2, 3, 4, 5)
    val doubled = numbers.map { it * 2 }
    val sum = doubled.reduce { acc, n -> acc + n }
    val evenNumbers = doubled.filter { it % 2 == 0 }
    println("Original numbers: ${numbers.joinToString()}")
    println("Doubled numbers: ${doubled.joinToString()}")
    println("Sum of doubled numbers: $sum")
    println("Even numbers: ${evenNumbers.joinToString()}")
}
```

```kotlin
data class Todo(val title: String, val description: String, var isCompleted: Boolean = false)

fun addTodo(): Todo {
    print("Enter the title of the todo: ")
    val title = readLine() ?: ""
    print("Enter the description of the todo: ")
    val description = readLine() ?: ""
    return Todo(title, description)
}

fun toggleTodoCompletion(todo: Todo) {
    todo.isCompleted = !todo.isCompleted
}
```
